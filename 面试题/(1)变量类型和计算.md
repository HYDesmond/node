- 4、JS变量按照存储方式区分为哪些类型，并描述其特点
> 考点：JS变量类型

> 知识点：
    - 值类型：将变量值储存在内存中（给变量赋值后，变量值不会改变）
    - 引用类型：变量指向一个指针（内存地址）

> 解答：
    -例子
    ```
        // 值类型
        var a = 100
        var b = a
        a = 200
        console.log(b) //100

        // 引用类型
        var a = {age:20}
        var b = a
        b.age = 21
        console.log(a.age) //21
    ```
